# Aristotelian Categorical Syllogistic Exhibit

## Formal identity

This exhibit implements the 256 categorical syllogism forms generated by four
statement types (`A`, `E`, `I`, `O`), three positions (major premise, minor
premise, conclusion), and four figures. Terms are `S`, `M`, and `P`.

Two semantic contracts are available:

- modern Boolean semantics, where universal statements have no existential
  import;
- traditional import semantics, where `S`, `M`, and `P` must each be nonempty.

`A(X,Y)` means every `X` is `Y`; `E(X,Y)` means no `X` is `Y`;
`I(X,Y)` means some `X` is `Y`; and `O(X,Y)` means some `X` is not `Y`.

## Complete semantics and decision procedure

Membership in `S`, `M`, and `P` produces exactly eight Venn regions. A finite
model is completely characterized, for this language, by the subset of regions
that are nonempty. There are therefore exactly `2^8 = 256` semantic models.

The decision procedure enumerates every model allowed by the selected import
semantics. A form is valid iff every model satisfying both premises satisfies the
conclusion. This enumeration is exhaustive, terminating, and complete—not a
bounded approximation.

The complete classification contains exactly 15 valid forms under Boolean
semantics and 24 under traditional import semantics. Tests enumerate all 256
forms under both contracts and preserve the strict containment relationship.

## Evidence objects

For a valid form, `Certificate.eligible_models` is an exact 256-bit set of every
premise-satisfying model. The checker independently reconstructs eligibility and
checks the conclusion in every member.

For an invalid form, `Certificate.countermodel` is a concrete Venn-region model.
The checker replays the import constraint, both premises, and the failed
conclusion. Missing, altered, or ineligible evidence is rejected.

All 512 form/semantics combinations generate evidence and replay it in unit tests.
Additional mutation tests alter exhaustive bitmaps and countermodels and require
rejection.

## Completeness contract

- The typed syntax represents every categorical form in the stated language.
- Boolean and traditional-import semantics are executable and total.
- Decision always terminates after at most 256 model checks.
- Every validity result carries exhaustive evidence.
- Every invalidity result carries a replayable countermodel.
- Canonical forms and fallacies are included as named regression cases.

This supports `verified_exhibit` at the repository’s `unit-tested` proof level.
It does not cover relational syllogisms, modal syllogistics, arbitrary natural-
language quantifiers, medieval supposition theory, or NLP parsing; those are
separate exhibits rather than silent extensions of this contract.
